男宿生活公約

電鍋、微波爐使用規定

男宿生活公約

如有和自治規定牴觸以自治規定為主

  1. 嚴禁帶非住宿生進入宿舍，違者依退宿辦理。
  2. 如違規點數有被誤記者，請在兩週內提出申請，若超過兩週則不受理。
  3. 白板及公佈欄只有各樓層幹部有權限使用，若對於白板或公佈欄有問題者，可向樓長進行討論，若直接對白板回覆者，記點乙次 。
  4. 請注意社群及公佈欄更新的公告，以免損失自己的權利（沒有不知者無罪）。
  5. 各房請自行訂房規；且不得與男宿舍生活公約及自治規定抵觸。
  6. 不得在宿舍內飼養任何動物，經勸說而未改善者記點三次。
  7. 值勤室信件及包裹，如放置超過一週（含）以上無人認領者，一律丟棄處理。
  8. 在公共區域（不含洗衣間）放置個人物品超過3天未認領，將一律丟棄處理。
  9. 私人垃圾一律不准丟棄於公共區域內（交誼廳、廁所、陽台…），如有發現，請向樓長、副樓長檢舉，被檢舉者，違者記點兩次 。
  10. 請勿跨樓層洗澡、丟垃圾，若發現請一律通知正、副樓，並且記點乙次 。
  11. 24：00時過後，勿在走廊大聲喧嘩、奔跑，以保持樓層安靜，經勸說而未改善者，記點乙次 。
  12. 交誼廳桌子使用後請保持乾淨，共同維護公共區域的整潔。
  13. 週一至週四及週日，電視觀賞時間至24：00；週五至週六，可延至02：00。
  14. 公用櫃及鞋櫃請按照床位順序放置。
  15. 寢室內電器用品除吹風機、電腦、手機及相機充電器外，嚴禁使用任何炊膳相關加熱爐具、電捲棒、離子夾等高用電壓器具，違者記點五次 。
  16. 若有寢室用電過度（同時使用多台吹風機）不幸斷電，請至值勤室找輔導老師協助， 且在寢室者均記點五次 。
  17. 使用脫水機、洗衣機、烘乾機及洗衣服，請在24：00之前完成工作；洗澡請於01:00前完成，違者記點乙次 。
  18. 廁所垃圾桶內請勿丟棄私人垃圾，違者記點兩次 ，打掃一週。
  19. 浴室內禁止使用吹風機，違者記點兩次 。
  20. 洗衣間無人認領的衣物，將在清冰箱時一同丟棄，不得異議。
  21. 簡報室使用時間為06：00至23：00。
  22. 湯汁、飲料、廚餘及殘渣請至烹飪室處理，勿倒入飲水機、洗手台及馬桶，如有違規，記點乙次 。

設備

使用規定

冰箱使用規定：

  1. 物品請自備標籤，並註明床號、姓名及放置日期（嚴禁填寫未來日期），違者一律丟棄，不另行通知。
  2. 放置冰箱的物品請使用便條紙寫上床號、姓名及當天放置日期（Ex：201-1 李小文 5/22），嚴禁填寫未來日期，若經發現一律丟棄。每周三為清冰箱日，請同學於20：50前更改完日期，21：00開始清冰箱。超過7天的物品也將一律丟棄。若在幹部清冰箱期間，要更改日期及補貼標籤者，將一律記點乙次 。
  3. 不得擺放生食及開封過手搖杯，例：蛋、生肉、菜（沙拉除外），需烹煮的一律不行擺放。如果蓄意丟棄過期食物、剩食及垃圾，違者記點兩次 ，不得異議。
  4. 若有藥物存放時，須提出申請，經輔導員核准後，另提供冰箱存放。
  5. 冰箱僅提供存放，不負保管責任，若丟私人垃圾，違者記點乙次 。
  6. 冰箱請勿拔掉插頭，違者記點五次 。

洗衣間使用規定：

  1. 洗衣機、烘衣機最後放入使用時間為23：00，超過24：00整還在運轉記點乙次 。
  2. 脫水機最後使用時間為23：50，超過24：00整還在運轉記點乙次 。
  3. 脫水機、洗衣機及烘乾機的使用方法已貼在牆上，請同學自行參閱，若使用不當，記點兩次 。

電鍋、微波爐使用規定：

  1. 簡易烹飪室設備使用時間為06：00至01：00（以前），遇有損壞請立即回報，違者記點兩次 。
  2. 微波爐禁止調至中強火以上及不得超過2分鐘。
  3. 使用完電鍋及微波爐後，請將插頭拔掉；當運轉停止後食品勿放置超過20分鐘以上。
  4. 使用電鍋時，請自備金屬容器或瓷器；使用微波爐時，請自備可微波之容器（嚴禁使用金屬器）。
  5. 每次使用後，請以抹布擦拭清理；並將鍋蓋或爐門打開通風，並拔掉插頭。

使用流程

電磁爐

  1. 插上插頭
  2. 放上鍋子
  3. 啟動開關
  4. 選擇模式
  5. 離開前請關上電源並拔除插頭
  6. 使用完畢請自行清潔

烤箱

  1. 插上插頭
  2. 開啟烤箱門
  3. 放入食物
  4. 選擇合適時間
  5. 烘烤時間尚未結束時請勿中途開啟
  6. 離開前請關上電源並拔除插頭
  7. 使用完畢請自行清潔並關上烤箱門

電鍋

  1. 插上插頭
  2. 開啟鍋蓋
  3. 放入食物
  4. 按下電鍋開關
  5. 蒸煮時間尚未結束時請勿中途開啟
  6. 離開前請關上電源並拔除插頭
  7. 使用完畢請自行清潔並蓋上鍋蓋

微波爐

  1. 插上插頭
  2. 開啟微波門
  3. 放入食物
  4. 選擇合適時間
  5. 微波時間尚未結束時請勿中途開啟
  6. 離開前請關上電源並拔除插頭
  7. 使用完畢請自行清潔並關上微博門

冰箱

  1. 打開冰箱
  2. 貼上便條紙（姓名、床號、日期）
  3. 放在合適位子
  4. 關緊冰箱門
  5. 若有液體溢出請自行清潔乾淨

洗衣機

  1. 添加洗劑
  2. 放入衣物（請勿超過7分滿）
  3. 關上機蓋
  4. 選擇洗程
  5. 投入錢幣20元
  6. 按啟動鍵開始洗衣
  7. 洗衣過程切勿開啟洗衣蓋
  8. 洗完後請確實拿取全部衣物

烘乾機

  1. 打開機門
  2. 放入衣物
  3. 關上機門
  4. 選擇溫度
  5. 投入錢幣20元
  6. 按啟動鍵開始烘衣
  7. 烘衣過程切勿開啟烘衣蓋
  8. 烘完後請確實拿取全部衣物

簡報室

使用簡報室電腦流程

  1. 開啟電腦主機
  2. 插入桌上HDMI線至主機
  3. 按下牆上down按鈕自動放下布幕
  4. 按下牆上on按鈕 開啟投影機
  5. 使用完畢按下牆上 up按鈕及off按鈕即可關閉
  6. 電腦關機並移除HDMI線使用完請歸放原位
  7. 椅子也請靠上桌子並關閉冷氣及點燈

使用自己電腦

  1. 插入桌上HDMI線至自己筆電
  2. 按下牆上down按鈕自動放下布幕
  3. 按下牆上on按鈕 開啟投影機
  4. 使用完畢按下牆上 up按鈕及off按鈕即可關閉
  5. 移除HDMI線使用完請歸放原位
  6. 椅子也請靠上桌子並關閉冷氣及點燈
